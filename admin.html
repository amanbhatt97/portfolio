Developer Console

Sign in with a GitHub access token to edit your portfolio content and resume. Changes are committed to amanbhatt97/portfolio and go live automatically.

How do I create a token?
  1. Open github.com → Settings → Fine-grained tokens (logged in as the repo owner).
  2. Name it portfolio-admin and pick an expiration (e.g. 1 year).
  3. Repository access → Only select repositories → choose portfolio.
  4. Permissions → Contents → Read and write.
  5. Generate, copy the token (github_pat_…) and paste it above.

This console is a static page — the token never leaves your browser except to call api.github.com. Anyone can open this page, but nothing can be changed without your token.
